# 📊 Personal Expense Tracker with Analytics

A full-stack Python web application that helps users track daily spending, visualize habits through dynamic charts, and manage accounts with a secure login system.

## 🚀 Features
* **User Authentication**: Secure Signup and Login functionality.
* **Smart Dashboard**: View spending summaries for the "Last 7 Days" and "Last 30 Days."
* **Data Visualization**: Automated bar charts showing daily spending trends using Matplotlib.
* **Expense Management**: Add and delete expenses with real-time total updates.
* **Database**: Persistent storage using SQLite and Flask-SQLAlchemy.

## 🛠️ Tech Stack
* **Backend**: Python (Flask)
* **Database**: SQLite
* **Frontend**: HTML5, CSS3 (Jinja2 Templates)
* **Data Science**: Matplotlib (for Graphing)

## 📥 Installation & Setup
1. Clone the repo: `git clone https://github.com/sanathoi109/programs.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python app.py`
4. Access at: `http://127.0.0.1:5000`
